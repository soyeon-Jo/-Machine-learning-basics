### 조건에 따라 행 추출

ex) 혈압이 70이상인 행만 추출하기
1. ```df['Age'] >= 70```은 각 행에서 Age가 70이상의 여부를 True/False Series로 반환한다.

2. 이 Series를 이용하여 ```df[df['Age']>=70]```
위와 같이 인덱싱하면 True인 행들만 추출 할 수 있다.


```python
print(type(df['BloodPressure'] >= 70)) #True/False로 반환되는지 확인
print((df['BloodPressure'] >= 70).head()) #앞부분을 확인
```

    <class 'pandas.core.series.Series'>
    0     True
    1    False
    2    False
    3    False
    4    False
    Name: BloodPressure, dtype: bool
    


```python
df[df['BloodPressure'] >= 70]
```

### 데이터 클리닝
대표적인 클리닝은 결측값(missing value)의 처리다.
<br>pandas는 결측값 여부를 반환하는 ```isnull``` 함수가 있다.<br>``` sum()``` 함수와 결합하면 각 열의 결측값 수를 알 수 있다:


```python
df.isnull().sum()
```

이외에
- `dropna()` - 결측치가 있는 모든 행을 없앤다.
- `fillna()` - 결측치를 정해진 값으로 바꾼다. 
가 있다.
